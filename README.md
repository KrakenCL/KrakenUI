# KrakenUI
JS Frontend for KrakenCL server.

Thanks Josephus Paye II for https://github.com/JosephusPaye/Keen-UI.

## Developing 
###Install node
```
$ brew install node
```

###Clone project
```
$ git clone git@github.com:KrakenCL/KrakenUI.git
```
###Prepare package
```
$ cd KrakenUI
$ npm install
```

###Launch development environment
```
$ npm run-script build
$ npm run-script dev
```
